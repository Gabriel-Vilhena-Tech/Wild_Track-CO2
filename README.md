#  Zoológico em C++

Um projeto em *C++* que simula o gerenciamento de um zoológico.  
Permite *cadastrar novos animais, **listar os existentes* e *exibir descrições* armazenadas em um arquivo de texto.

---

## Funcionalidades

- *Cadastrar* novos animais  
- *Listar* todos os animais cadastrados  
- *Buscar* animal pelo nome  
- *Atualizar* ou *remover* registros existentes  

---

##  Armazenamento

Todas as informações são gravadas no arquivo  
data/zoologico.txt, garantindo que os dados permaneçam salvos mesmo após o encerramento do programa.
